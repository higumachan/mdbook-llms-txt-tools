# 第2章: 機能紹介

この章では、mdbook の高度な機能をいくつか紹介します。

## シンタックスハイライト

様々な言語のコードブロックに対応しています：

```rust
fn main() {
    println!("Hello from Rust!");
}
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}
```

## 検索機能

ページ上部の検索バーを使用することで、ドキュメント全体を検索することができます。

## カスタマイズ

`book.toml` ファイルを編集することで、様々な設定をカスタマイズできます：

- テーマの変更
- プラグインの追加
- 出力形式の設定
- その他の設定
